# 💳 Digital Wallet Analytics Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **Digital Wallet Analytics Dashboard** built using Microsoft Excel. It analyzes transaction data to uncover key business insights related to revenue, user behavior, payment methods, and product performance.

The dashboard is designed to help stakeholders quickly understand performance trends and make data-driven decisions.

---

## 📊 Dashboard Structure

The project consists of **two main dashboards**:

### 1️⃣ Transaction Overview Dashboard

Focuses on overall business performance:

* Total Revenue & Net Revenue
* Total Transactions
* Average Revenue
* Total Cashback & Fees
* Monthly Revenue Trend
* Transaction Status Distribution (Successful, Failed, Pending)
* Payment Method Analysis
* Device Type Distribution
* Top 10 Categories by Transaction Count

### 2️⃣ User & Product Insights Dashboard

Focuses on deeper analysis:

* Top 10 Product Categories by Paid Amount
* Top 10 Products by Paid Amount
* Revenue & Transactions by User
* Revenue Distribution by Location (Urban, Suburban, Rural)
* Fees by Payment Method

---

## 📈 Key Insights

* The majority of transactions are **successful**, indicating strong system performance.
* **Urban locations** contribute the highest share of total revenue.
* **Android devices** dominate transaction activity.
* Categories such as **Streaming Services and Bills** generate the highest revenue.
* **Bank Transfer and Debit Cards** are the most frequently used payment methods.
